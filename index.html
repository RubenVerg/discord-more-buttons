Custom Commands
This server's bots which support custom commands are:
MEE6

Prefix: ! Quick commands:

  • !youtube <search>
    Search in YouTube
  • !start-quiz
    Start a music quiz
  • !record
    Record your voice in a MP3 file
  • !rank
    See your rank

YAGPDB

Prefix: - Quick commands:

  • -ticket open <reason>
    Open a ticket
  • !start-quiz
    Start a music quiz
  • -help
    See help
